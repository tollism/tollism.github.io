joules = *4.18; 1 calorie = 4.18 Joules Use cases: Finding the specific heat capacity of two objects cooled in eachother [with one known]: qb = -q[known] --> m(dT) * c_b = m(dT) * c_a: c_b = c_a * m(dT)/m(dT) '''bond Enthalpies''' = Bonds broken [reactants] - Bonds formed [products] '''dH = Sum[bonds broken] + Negative Sum[bonds formed]. '''Moles from added heat''' = (Added Heat * Coefficient of Product) / dH of reaction. '''Moles from half reactions''' = Find target of product/missing info. Reversing the sum if necessary. EX: * dH of CO(g) + 2H₂(g) → CH₃OH(g) * 2CO(g) + O2(g) → 2CO2 (g)  ΔH₁ = −566 kJ * 2H2 (g) + O2 (g) → 2H2O (g)  ΔH₂ = −571 kJ * 2 CH3OH (g) + 3O2 (g) → 2CO2 (g) + 4H2O (g)  ΔH₃ = −1430 kJ We have the enthalpy for 2CO = so multiply the whole base reaction by two. We need 4H2 = so multiply 2 by 3. CH3OH needs to be on the product side - so we can flip the enthalpy of step 3 to get +1430 KG 2CO + 4H2 -> 2CH3OH Then divide by two to get 1 CH3OH (-566 + 2[-571] + 1430 )/2 = -139Kj >
